BLUCE ATENDANCE

Bluce is a Bluetooth attendance web application that utilizes modern technology to make attendance tracking more efficient and convenient. The application allows users to register their mobile devices as attendance markers, making it easy for them to mark their attendance with a simple tap. The Scope of this proposed project is to enable attendance tracking through the use of Bluetooth technology. This includes the ability to register users, manage attendance records, generate reports, and integrate with other systems. 


The audience of this system will be: 
1. Students 
2. Administrators

FEATURES

Bluce offers a range of features to help organizations and educational institutions manage attendance effectively. Some of our key features are : 


1. Bluetooth Connectivity
                
2. Real-time Attendance Monitoring
                
3. Customizable Attendance Reports
                
4. Automatic Attendance Notifications
                
5. User-friendly Interface
